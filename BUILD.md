# How to build
1. Run configure script, do this if you reinstall QT.
    ```bash
    ./configure_auto_build
    ```
    - If you get an error, run the command
        ```bash
        chmod a+x configure_auto_build; ./configure_auto_build;
        ```
2. Run this command to build qgroundcontrol.
   ```bash
   ./build.sh
   ```
